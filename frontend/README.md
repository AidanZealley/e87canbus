# Frontend

React UI used for the development simulator and, later, the in-car touchscreen.

- `src/application/` — top-level composition and connection lifecycle.
- `src/dashboard/` — vehicle-facing status and configuration screens.
- `src/simulator/` — virtual controls, CAN trace, and simulator-specific types.
- `src/api/` — HTTP and WebSocket clients.
- `src/components/ui/` — reusable presentation primitives.
